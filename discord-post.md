Hey OpenClaw community! 👋

Tired of your agent forgetting everything when you start a new session? **claw-git-memory** is here to solve that!

## ✨ What it does

- **Remember conversations** across sessions
- **Full version history** with Git
- **Sync across devices** via GitHub/GitLab
- **Works offline** - no cloud required

## 🚀 Quick Start

```bash
clawhub install claw-git-memory
```

Then follow the README to set up your `MEMORY.md` and start tracking your agent's memory with Git!
